Para rodar o projeto é necessário alterar o arquivo "\e-Auditoria API\Infrastructure\DependencyInjection.cs" e alterar o a string de conexão com o banco de dados no método AddInfrastructure(). 
Em seguida basta basta abrir a pasta "\e-Auditoria API\WebUI" com CMD e executar dotnet run. A Api esta propriamente compativel com swagger, então após o dotnet run, abra
o endereço em que esta sendo hospedado e adicione "/Swagger" ao final. Devido a minha falta de experiência na área e prazo eu não consegui fazer um front end adequado, porém
caso aceitem, acredito que terei ele pronto até terça feira que vem ( 28/06/2022 ) onde enviarei o projeto novamente propriamente atualizado.
